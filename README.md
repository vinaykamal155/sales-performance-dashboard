# Sales Performance Dashboard

## Overview
This project presents an interactive Tableau dashboard analyzing e-commerce sales performance from January to March 2026.

## Tools Used
- SQL (Data extraction & transformation)
- Excel (Exploratory analysis & pivot tables)
- Tableau Public (Dashboard development)

## Key Insights
- Electronics category contributes the majority of total revenue.
- February recorded the highest monthly sales.
- Mobile is the top-performing product by revenue.
